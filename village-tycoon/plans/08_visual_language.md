# 08. Visual Language

UI-палитра, типографика, паттерны интерфейса. Всё что не игровой канвас, а **окружающие** игровые элементы (HUD, меню, модалки, тултипы).

## Inspiration / референсы

- **Clash of Clans** — главный визуальный референс
- **Hay Day** — палитра, тёплая атмосфера
- **Township** — макеты экранов
- **Forge of Empires** — структура меню
- **Diablo Immortal** — тонкие подсказки на тач-интерфейсе

## Типографика

### Семейство шрифтов

- **Primary**: `'Fredoka One'`, `'Lilita One'`, или похожий жирный округлый sans-serif
- **Secondary** (числа, статы): `'Inter'`, `'Roboto'` — более нейтральный
- **Numbers** (большие счётчики): тот же `'Fredoka One'` для consistency

Шрифты подключаем через Google Fonts или скачиваем в `public/fonts/` (для офлайна).

### Размеры

```css
--text-xs:   12px;   /* подписи, тултипы */
--text-sm:   14px;   /* вторичный текст */
--text-md:   16px;   /* основной */
--text-lg:   20px;   /* заголовки секций */
--text-xl:   28px;   /* заголовки модалок */
--text-2xl:  36px;   /* счётчики ресурсов */
```

### Эффекты текста

- **Outline**: 2px тёмная обводка (`text-shadow: 2px 2px 0 #4A2F1A`) — на всех заголовках и кнопках, как в CoC
- **Drop shadow**: лёгкая тень для всего текста на ярких фонах
- Никогда не использовать тонкий шрифт без обводки на цветном фоне

## UI компоненты

### 1. Кнопка (button)

Стиль: **деревянная пластина с золотыми углами**.

```
┌─────────────────────────┐
│    [icon]   ПОСТРОИТЬ   │   ← warm wood texture
└─────────────────────────┘   ← shadow
```

- Высота: 56px (mobile), 48px (desktop)
- Border-radius: 12px
- Gradient: top lighter wood → bottom darker wood
- Outline: 2px `--wood-dark`
- Hover/Active: лёгкий scale(0.97) + brighter highlight
- Press: моментальная haptic (через Telegram WebApp API)

Варианты:
- **Primary** (стандартный, дерево) — для основных действий
- **Premium** (золото) — для покупок за гемы
- **Danger** (красный) — для destructive (снести)
- **Disabled** (серый, прозрачность 50%) — недоступно

### 2. Resource counter (счётчик ресурсов)

Маленький "чип" с иконкой ресурса и числом:

```
   ┌───────────────────┐
🪵 │  1,234 / 5,000    │
   └───────────────────┘
```

- Высота: 36px
- Background: `--bg-elevated` с лёгким градиентом
- Border-radius: 18px (pill)
- Icon (иконка ресурса) слева, размер 32px, выходит за границу чипа
- Число — Fredoka One, цвет ресурса
- При приближении к капу: число становится оранжевым, при capped — красным
- Animation: при изменении числа — короткий count-up (300ms)

### 3. Modal panel

Большое окно поверх затемнённого фона.

```
╔═══════════════════════════════╗
║   ⊗     ЛЕСОПИЛКА УР. 2      ║   ← header с close-кнопкой
╠═══════════════════════════════╣
║                               ║
║   [building image preview]    ║   ← preview
║                               ║
║   Производит: 🪵 250/час     ║
║   Жители: 0/2                 ║
║                               ║
║   ┌──────────────────────┐    ║
║   │   ПРОАПГРЕЙДИТЬ       │    ║   ← primary CTA
║   └──────────────────────┘    ║
║   Стоимость: 🪵500 🪨200    ║
║                               ║
╚═══════════════════════════════╝
```

- Background: `--bg-secondary` с deep shadow
- Border: 4px `--wood-dark`, скруглённые углы
- Header: золотой акцент, текст крупный
- Внутри padding 16-24px

### 4. Bottom navigation

5 круглых кнопок внизу экрана.

```
┌───────────────────────────────────┐
│                                   │
│         [game canvas]             │
│                                   │
└───────────────────────────────────┘
  🏗️    🔬    📜    👥    🛒
 BUILD  TECH QUEST FRIENDS SHOP
```

- Высота: 80px
- Каждая кнопка: 64×64
- Активная — slight bounce + glow
- Notification badge (красная точка) если есть непросмотренные события

### 5. Top resource bar

Горизонтальная панель сверху с 5 главными ресурсами.

```
┌─────────────────────────────────────┐
│ 🪵 1.2k  🪨 800  🍞 2k  💧 500  🪙 ▶  │
└─────────────────────────────────────┘
```

- 5 видимых ресурсов + кнопка "▶" открывает все
- Высота: 48px
- Каждый чип кликабельный — открывает инфо о ресурсе

### 6. Notification toast

Всплывающее уведомление снизу-сверху.

```
        ┌──────────────────────────┐
        │  ✓  Лесопилка построена  │
        └──────────────────────────┘
```

- Position: top center (под top bar) или bottom center (над bottom nav)
- Анимация: slide-in 300ms, hold 3s, slide-out 300ms
- Цвет фона по типу: green/red/blue/orange

### 7. Tooltip

Маленькая подсказка при tap-and-hold или hover.

```
   ┌──────────────────────┐
   │ Дерево               │
   │ Нужно для построек.  │
   │ Текущий кап: 5,000   │
   └──────────────────────┘
              ▼
```

- Background: `--bg-primary`, светлая обводка
- Text: small, секундарный цвет
- Появляется через 500ms hold

### 8. Quest card

Карточка квеста в списке.

```
┌────────────────────────────────────┐
│  📜 Построй свою первую лесопилку │
│  ─────────────────────────────     │
│  ████████████████░░░░░░  60%      │
│                                    │
│  Награда: 🪙 50  ⭐ +100xp         │
│  ┌────────────────┐                │
│  │   ЗАБРАТЬ      │                │
│  └────────────────┘                │
└────────────────────────────────────┘
```

- Прогресс-бар обязателен для всех квестов
- "Забрать" кнопка появляется когда выполнен

## Анимации

### Принципы

1. **Bounce / squash-stretch** для всего интерактивного — кнопки слегка "плющатся" при клике
2. **Pop-in** для появления элементов (scale 0 → 1 с overshoot 1.1 → 1)
3. **Count-up** для всех чисел при изменении
4. **Particles** при значимых событиях:
   - Постройка завершилась → залп звёзд из неё
   - Ресурс получен → летящая иконка ресурса в счётчик
   - Уровень повысился → конфетти
5. **Подсказки-сияния** для важных действий ("кликни сюда")

### Длительности

```javascript
const ANIM = {
  fast:    150,   // hover, tap response
  medium:  300,   // toast, modal in/out
  slow:    600,   // count-up чисел
  reward:  1200,  // получение награды
  major:   2000,  // level up, era advance
};
```

### Easing

- Default: `cubic-bezier(0.34, 1.56, 0.64, 1)` — bouncy
- Linear для прогресс-баров и таймеров
- Ease-out для появлений
- Ease-in для исчезновений

## Haptic feedback (через Telegram WebApp API)

```javascript
window.Telegram.WebApp.HapticFeedback.impactOccurred('light');    // tap кнопки
window.Telegram.WebApp.HapticFeedback.impactOccurred('medium');   // постройка
window.Telegram.WebApp.HapticFeedback.impactOccurred('heavy');    // награда
window.Telegram.WebApp.HapticFeedback.notificationOccurred('success'); // квест
window.Telegram.WebApp.HapticFeedback.notificationOccurred('error');   // ошибка
```

## Звук (опционально, MVP без)

В MVP **звука нет**. Telegram-игры часто открывают в общественных местах, звук бесит. Добавим в Beta с toggle-кнопкой, off по умолчанию.

## Адаптивность

Целевые breakpoints:

| Платформа | Ширина | Поведение |
|---|---|---|
| Mobile portrait | 360-450px | Default, всё оптимизировано |
| Mobile landscape | 600-900px (короткое) | Уменьшенные UI-панели |
| Tablet portrait | 700-900px | Те же макеты, чуть больше padding |
| Desktop | 1280+px | Игровое поле максимум 480px ширина (как у CoC), остальное — фоновая виньетка |

## Тёмная / светлая тема

Telegram WebApp передаёт `colorScheme: 'dark' | 'light'`. **Мы используем только тёмную** — соответствует мульту с насыщенными цветами. Светлая тема в idle-tycoon играх редко выглядит хорошо. Принудительно ставим `dark` независимо от Telegram-темы.

## Карта Z-индексов

```
0    : фон (тайлы, deco)
10   : здания
15   : строящиеся здания (полупрозрачные)
20   : NPC, юниты
30   : эффекты на канвасе (частицы)
100  : top resource bar
110  : bottom navigation
200  : modals (overlay)
210  : modal content
300  : toasts
400  : tooltips
500  : critical alerts (level up, era advance)
```

## Что нельзя

- Тонкие линии < 2px (на retina пиксели сольются)
- Чисто чёрный текст на чисто белом фоне (слепит, не в стиле)
- Большие плоские заливки без текстуры (выглядит как Excel)
- Иконки ресурсов разных стилей рядом
- Полупрозрачные кнопки (нечитаемо в Telegram-фоне)

## Что нужно

- Контраст текста к фону минимум 4.5:1 (WCAG AA)
- Тачи минимум 44×44 px (Apple HIG)
- Все интерактивные элементы — с явным feedback (visual + haptic)
- Числа всегда с разделителями тысяч (1,234,567)
- Длинные числа форматировать (1.2M, 500k) — у нас уже есть `fmtCompact` в казино, переиспользуем
